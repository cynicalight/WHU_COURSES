### 1.4.2
保存图像使用jpg会被压缩，读取加密信息一直不准确，即使图像质量设置为100也没用。

```python
cv2.imwrite("xx.jpg", img, [cv2.IMWRITE_JPEG_QUALITY, 100])
```

最终使用 png 格式保存图像后，读取加密信息成功。
- 可以通过增大 alpha 来提高鲁棒性，抗压缩
